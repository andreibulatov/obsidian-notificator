# Obsidian Notificator

Notificator — это плагин для Obsidian, который позволяет создавать системные уведомления, привязанные к вашим заметкам. Он поддерживает интеграцию с операционными системами macOS, Windows и Android, помогая вам не пропускать важные события и задачи, описанные прямо в Markdown.

## Основные возможности

- **Системные уведомления**: Плагин отправляет стандартные оповещения ОС (через Web Notifications API).
- **Привязка к контексту**: Уведомление ведет прямо к той заметке и тому месту, где оно было создано.
- **Интерактивный виджет**: Удобный выбор даты и времени с помощью встроенного календаря и поля для комментария.
- **Система статусов**: Отслеживание жизненного цикла уведомления: `ok`, `missed` (пропущено), `cancel` (отменено), `postponed` (отложено).
- **История изменений**: Автоматическая запись всех изменений (создание, перенос времени) прямо в блок кода уведомления.
- **Calendar View**: Отдельное представление с календарем для просмотра всех запланированных и прошедших уведомлений.
- **Фоновый планировщик**: Ежеминутная проверка актуальных уведомлений.

## Использование

Для создания уведомления используйте специальный блок кода `notifiactor`:

```notifiactor
20.04.2026 15:40 Комментарий к уведомлению
-- history
20.04.2026 10:00 created notification - 20.04.2026 15:40
```

Вы можете нажать на иконку или использовать команды плагина для вызова виджета редактирования.

## Разработка и сборка

### Предварительные требования

- Установленный [Node.js](https://nodejs.org/) (рекомендуется LTS версия).
- Obsidian.

### Установка зависимостей

```bash
npm install
```

### Сборка проекта

Для сборки плагина (создания файла `main.js`):

```bash
# Режим разработки (с inline sourcemaps)
node esbuild.config.mjs

# Продакшн сборка
node esbuild.config.mjs production
```

## Установка в Obsidian

1. Создайте папку для плагина в вашем хранилище: `.obsidian/plugins/obsidian-notificator`.
2. Скопируйте в эту папку следующие файлы:
    - `main.js` (после сборки)
    - `manifest.json`
    - `styles.css` (если имеется)
3. Перезагрузите Obsidian или обновите список плагинов в настройках.
4. Включите плагин **Notificator** в разделе "Community plugins".
