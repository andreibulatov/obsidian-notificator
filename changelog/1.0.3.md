# Changelog

## [1.0.3] - 2026-04-20

### Добавлено
- В представление уведомлений добавлен полноценный календарный вид (сетка на месяц).
- Реализована навигация по месяцам в календаре.
- Отображение уведомлений в ячейках календаря с возможностью быстрого перехода к заметке.
- Сохранена возможность просмотра уведомлений в виде списка под календарем.
