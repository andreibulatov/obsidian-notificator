# Changelog

## [1.0.0] - 2026-04-20

### Добавлено
- Создан плагин Notificator для Obsidian.
- Поддержка системных уведомлений для Mac OS, Windows и Android (через Web Notifications).
- Парсер блоков кода `notifiactor` для хранения данных уведомления и истории в Markdown.
- Виджет (модальное окно) для создания и редактирования уведомлений с выбором даты и времени.
- Система статусов уведомлений: `ok`, `missed`, `cancel`, `postponed`.
- Автоматическое ведение истории изменений внутри блока уведомления.
- Представление "Calendar View" для визуализации всех уведомлений в календаре.
- Фоновый планировщик для ежеминутной проверки и отправки уведомлений.
